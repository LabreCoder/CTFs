## 🎯 Objetivo desse Repositório é registrar os meus CTFs do THM

### 1. Divisão desse repositório

Ele será dividio em diversos CTFs pertencentes à plataformas conhecidas, exemplo: 

- TryHackMe ou THM: <i>[TryHackMe](https://tryhackme.com/)</i>
- Hack The Box: <i>[HackTheBox](https://www.hackthebox.com/)</i>
- HackerDNA: <i>[HackerDNA](https://hackerdna.com/)</i>
- Entre outros...

<hr>

### 2. Como irá funcionar dentro de cada pasta?

A princípio irei deixar uma pasta para cada site de CTFs e dentro de cada uma dessas pastas irá conter os nomes dos exercícios:

- takeover
- neighbour

Além disso, dentro de cada um dos exercícios terá a seguinte divisão:

- README.md
    > a fim de entender o objetivo e resultado desse CTF

- report
    > contendo um relatório gerencial e um relatório técnico

- images
    > apresentando as evidências que eu registrei durante a resolução do exercício, sem compartilhar a <b>flag</b>, pois não quero ela seja copiada e colada, porém testada.

<hr>
